# Auralane-your-e-commerce

Live e-commerce web application built using React, TypeScript, and Supabase with authentication, product management, cart, and checkout features.

## Tech Stack
- React
- TypeScript
- Supabase
- Tailwind CSS

## Features
- Secure authentication
- Product listing
- Cart & checkout flow
- Responsive UI

## Setup
npm install  
npm run dev
